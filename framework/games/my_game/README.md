## Example game fixture (`my_game`)

Minimal Java + web client scaffold used for local CLI smoke tests. Not published to production.

- Run `gamedev-cli test`
- Run `gamedev-cli build`
- Run `gamedev-cli login --display-name you --password <secret>` (or `login --user-id <uuid>` with a publish token for deploy-only flows)
- Run `gamedev-cli deploy --auto-publish`
